📝 Mac 使用方法：

  1. 复制上面的链接到浏览器 → 下载 psd-to-prefab-installer.sh
  2. 打开终端，进入下载目录：
     cd ~/Downloads
  3. 运行脚本：
     chmod +x psd-to-prefab-installer.sh
     ./psd-to-prefab-installer.sh
  4. 脚本会在当前目录生成 packages/psd-to-prefab/ 完整目录
  5. 把 packages/psd-to-prefab/ 复制到 Cocos Creator 项目的 packages/ 下
  6. cd 你的项目/packages/psd-to-prefab && npm install
  7. 重启 Cocos Creator
